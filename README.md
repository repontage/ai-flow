# AI Flow

> 노드 기반 시각적 AI 워크플로우 자동화 플랫폼

여러 AI API를 드래그앤드롭으로 연결해 자동화 파이프라인을 구성하는 로컬 우선(Local-first) 웹 애플리케이션입니다. n8n에서 영감을 받아 제작되었습니다.

![image](https://github.com/user-attachments/assets/placeholder)

---

## 주요 기능

- **시각적 워크플로우 편집기** — 드래그앤드롭으로 노드를 캔버스에 배치하고 연결
- **멀티 AI 프로바이더** — OpenAI, Anthropic Claude, Google Gemini, Groq, Ollama를 하나의 워크플로우에서 조합
- **로컬 우선 보안** — API 키는 브라우저 localStorage에만 저장, 서버로 전송되지 않음
- **실시간 실행 로그** — 노드별 실행 상태, 입출력, 소요 시간 확인
- **JSON 내보내기/가져오기** — 워크플로우를 파일로 공유
- **빠른 시작 템플릿** — 자주 쓰는 패턴을 즉시 사용

## 내장 템플릿

| 템플릿 | 설명 |
|--------|------|
| 📄 AI 문서 요약기 | 텍스트 입력 → AI 요약 → 표시 |
| 🌐 다국어 번역기 | 한국어·일본어 동시 번역 (병렬 실행) |

## 기술 스택

- **React 18** + TypeScript
- **@xyflow/react** — 노드 에디터 캔버스
- **Zustand** — 상태 관리 + localStorage 영속성
- **TailwindCSS** — 스타일링
- **Vite** — 빌드 도구

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

브라우저에서 `http://localhost:3000` 접속

## 사용 방법

1. **설정**에서 사용할 AI 프로바이더의 API 키 입력
2. **새 워크플로우** 생성 또는 템플릿 선택
3. 왼쪽 팔레트에서 노드를 캔버스로 드래그
4. 노드를 클릭해 오른쪽 패널에서 설정 구성
5. 노드 핸들을 드래그해 연결
6. **실행** 버튼으로 워크플로우 실행

---

## 템플릿 변수 문법

노드 설정의 프롬프트 템플릿에서 이전 노드의 출력을 참조할 수 있습니다.

```
{{input}}                        → 직전 노드의 출력
{{nodes.nodeId.output}}          → 특정 노드의 출력
{{nodes.nodeId.output.text}}     → AI 노드의 텍스트 출력
{{now}}                          → 현재 시각 (ISO 8601)
{{timestamp}}                    → Unix timestamp
```

## 보안

API 키는 브라우저의 `localStorage`에만 저장됩니다. 서버로 전송되거나 외부에 노출되지 않으며, AI API 호출 시에만 메모리에서 사용됩니다. 공유 컴퓨터에서 사용 후에는 설정에서 키를 삭제하세요.
